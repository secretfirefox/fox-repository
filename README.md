Welcome to my personal repository, the Fox Repository!

The purpose of this repo is to make it easier setting up Void Linux with my special set of customizations, to arrive always in the same desktop and get a system up and running faster through shell scripts. 

This script will give you a GNOME installation of Void Linux.

This aims to be as plug-and-play and functional as the likes of Linux Mint, except with a much lighter and concise init-system and without some user-friendly additions which are not as necessary for a medium-level user. Regardless, this does not mean this system will be harder to use as GNOME provides great functionality out of the box, and with this being a minimal install, it should work great for any computer with at least 4GB of RAM as it will use at most 500MB of RAM at boot. Not like some other ready to go systems like Tumbleweed, which I have seen use 1.6GB at boot even on a system with only 2GB of RAM. Yikes.

Runit and Void Linux clearly allow you to save resources and leave more room for your games to run adequately, without it all getting taken over by SystemD as PID 1. 

So, if you are ready to start a new experience, begin with the installation scripts. Remember, the flatpak script should be launched manually only after you have logged in the desktop session. This is to avoid issues with polkit authentication. Enjoy!
